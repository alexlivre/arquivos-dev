# Antigravity Agent Skills

Este repositório contém uma coleção de skills especializadas para o **Antigravity Agent**, projetadas para potencializar o desenvolvimento de páginas web com foco em Copywriting, SEO e UI/UX.

## 🚀 Instalação

Para instalar as skills em seu projeto, utilize o script `install_antigravity_agent.cmd` incluído neste repositório.

1. Execute o arquivo `install_antigravity_agent.cmd`.
2. O script validará a estrutura dos arquivos de origem.
3. Quando solicitado, insira o **caminho completo** para a raiz do seu projeto (onde a pasta `.agent` deve ser criada ou atualizada).
4. O script copiará automaticamente as skills para o diretório `.agent/skills` do seu projeto.

> **Nota:** O arquivo `INKPAGE_UI_KIT.md` também será copiado para a raiz do seu projeto, contendo todos os componentes visuais padrão (Botões, Cards, etc).

## 📦 Skills Disponíveis

### 1. Page Copywriter Pro (`page-copywriter-pro`)
**Foco:** Criação de textos persuasivos e focados em conversão.
- Cria copy para landing pages, páginas de vendas, home, serviços, etc.
- Estrutura hierarquia de texto (H1, H2, CTAs).
- Utiliza gatilhos mentais e psicologia comportamental.
- **Idioma:** PT-BR.

### 2. SEO Landing Page Specialist (`seo-landing-page-specialist-ptbr`)
**Foco:** Otimização para motores de busca (Google).
- Define títulos e meta descriptions otimizados.
- Planeja a estrutura de headers (H1, H2, H3).
- Sugere palavras-chave e links internos.
- Entrega um "SEO Pack" pronto para implementação.
- **Idioma:** PT-BR.

### 3. UI/UX Landing Page Pro (`ui-ux-landing-page-pro`)
**Foco:** Design visual, layout e experiência do usuário.
- Traduz o texto em wireframes e estruturas visuais.
- Define paletas de cores, tipografia e hierarquia visual.
- Foca em usabilidade e otimização de conversão (CRO).
- Trabalha em conjunto com as skills de Copywriting e SEO.
- **Tech Stack:** Impõe o uso de Tailwind CSS via CLI para garantir otimização (CSS purgado).

### 4. Web Performance Optimizer (`web-performance-optimizer-no-layout-ptbr`)
**Foco:** Otimização de performance (Core Web Vitals) sem alterar layout.
- Melhora LCP, INP e CLS.
- Otimiza imagens, fontes, cache e entrega de assets.
- Não altera o design visual ("invisible optimization").
- **Idioma:** PT-BR.
